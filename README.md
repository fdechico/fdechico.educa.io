## Serviço de Ensino Remoto Gratuito

## Descrição do Projeto
O Serviço de Ensino Remoto Gratuito é uma plataforma web destinada a oferecer cursos e materiais educativos acessíveis a todos. O projeto busca atender ao Objetivo de Desenvolvimento Sustentável (ODS) relacionado à educação de qualidade e à promoção da igualdade de oportunidades.

## Funcionalidades
- Navegação Intuitiva: Menus de navegação simples para acessar cursos, cadastro e contato.
- Cadastro de Usuário: Formulário para registro de novos usuários com validação de dados.
- Cursos Disponíveis: Listagem de cursos em diversas áreas do conhecimento.
- Depoimentos: Espaço para feedback de usuários, promovendo a interação e a confiança na plataforma.
- Modo Escuro: Alternância entre tema claro e escuro para melhor experiência do usuário.

## Como os Usuários Podem Utilizá-lo
1. Acessar a Plataforma: Visite o link da página no GitHub Pages.
2. Cadastrar-se: Clique na opção de cadastro e preencha os campos obrigatórios.
3. Explorar Cursos: Navegue pelos cursos disponíveis e clique nos links para mais informações.
4. Deixar Depoimentos: Após concluir um curso, os usuários podem deixar feedback para ajudar outros alunos.
